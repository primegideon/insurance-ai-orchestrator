Startup Lifespan Singleton

InsuranceRiskEvaluator is built once inside the lifespan context manager — not on every request. This matters because the IBM SDK authenticates with IAM at construction time, so instantiating per-request would add a network round-trip and cost to every call.

Routes Endpoints
GET / Health check — returns {"status":"ok"}
POST /api/v1/evaluate-claim Accepts claim + profile, returns RiskEvaluationReport
The /api/v1/ prefix keeps the path versioned so a v2 can coexist without breaking existing frontend calls.
Errors HTTP Error Mapping
502 Bad Gateway — raised when ValueError is caught, meaning the LLM returned malformed or unparseable JSON. The fault is upstream (the AI model), not the client.
503 Service Unavailable — raised when RuntimeError is caught, meaning the IBM watsonx API call itself failed (auth, quota, network timeout).
CORS CORSMiddleware — Allowed Origins

Three localhost origins are whitelisted to cover every common frontend dev server. Add your production domain here before deploying.

  • http://localhost:3000 — React / Next.js (Create React App, Next dev)
  • http://localhost:5173 — Vite (React, Vue, Svelte via Vite)
  • http://localhost:8080 — Vue CLI / generic fallback
To run the server: uvicorn app.main:app --reload from the backend/ directory. Then visit http://localhost:8000/docs for the interactive Swagger UI.
